      开发注意事项
      配置文件读取的总是与 index.html 同级的 ./static/configuration.json , 而修改配置的时候是通过 github-api 修改 你的用户名.github.io 下的 /static/configuration.json, 所以本地 npm run dev 的时候会出现修改配置无效的情况, 如果需要修改的话修改本地的配置文件就行, 发布到 github 之后不影响
